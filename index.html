Combinatorics

Combinatorics is the branch of Mathematics dealing with the study of finite or countable discrete structures. It includes the enumeration or counting of objects having certain properties.
https://www.geeksforgeeks.org/mathematics-combinatorics-basics/

Example:Finding power of a factorial divisor 

                int fact_pow (int n, int k) {
                    int res = 0;
                    while (n) {
                        n /= k;
                        res += n;
                    }
                    return res;
                }
